# 📩 SMS API Integration (PHP)

Send SMS messages in PHP using the **MobileSasa API**. This project includes scripts for sending **bulk personalized SMS** and **single messages**.

---

## 🚀 Features

- Send **bulk personalized SMS** using PHP cURL  
- Send **single SMS messages** using HTTP client (Guzzle)  
- Easy to automate notifications (billing, reminders, alerts)  

---

## 📂 Project Files

### `sms2.php`
- Sends bulk personalized messages  
- Uses PHP cURL  
- Customize messages per recipient  

### `Sms.php`
- Sends a single message  
- Uses HTTP client (e.g., Guzzle)  
- Can loop through multiple recipients  

---

## 🛠️ Requirements

- PHP 7+  
- cURL enabled  
- Internet connection  
- MobileSasa API Key  
- Composer (optional, for HTTP client like Guzzle)
